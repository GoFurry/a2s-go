# a2s-go Roadmap

## Open Tasks

### Phase 4

Goal: continue expanding real-world compatibility coverage.

Remaining work:

1. Build and maintain a compatibility matrix from real server targets.
2. Validate older game and non-mainstream server compatibility.
3. Validate unusual challenge refresh behavior on live servers.
4. Validate compressed packet behavior on live servers.
5. Record compressed-packet trigger patterns by query type, engine generation, and payload size so we can estimate real-world prevalence instead of relying on protocol assumptions.
6. Validate split packet behavior on live servers.
7. Persist live regression outcomes into release history.
8. Accumulate older or non-mainstream server samples.
9. Accumulate real challenge, split, and compressed response samples.
